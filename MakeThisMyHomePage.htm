How to Make BoardPark.com Your HomePage

Internet Explorer 5.0:
1. In the Tools menu, click Internet Options.
2. Click the General tab.
3. The Home Page option is at the top of the screen. T ype http://www.boardpark.com into the Address text box.
4. Click OK.

Internet Explorer 4.0:
1. In the View menu, click Internet Options.
2. In the Home page section, type http://www.boardpark.com/ into the Address text box.
3. Click OK.

Internet Explorer 3.0 for Windows:
1. In the View menu, click Options and choose the Navigation tab.
2. In the Page text box, choose Start Page.
3. In the Address text box, type http://www.boardpark.com/
4. Click OK.
5. Now click the Home button in the toolbar.

Internet Explorer 3.0 for Macintosh:
1. On the Edit menu, click Preferences.
2. In the list on the left of the dialog box, click the Home/Search button (under Web Browser).
3. In the Address text box, type http://www.boardpark.com/
4. Click OK.
5. Now click the Home button in the button bar.

Internet Explorer 4.0 for Macintosh:
1. On the Edit menu, click Preferences.
2. In the list on the left of the dialog box, click the Home/Search button (under Web Browser).
3. In the Address text box, type http://www.boardpark.com/
4. Click OK.
5. Now click the Home button in the button bar.

Netscape Communicator:
1. In the Edit menu, click Preferences.
2. Click the "Navigator Starts with Home Page" button in the upper box.
3. In the "Location:" text box in the middle of the window, type http://www.boardpark.com
4. Click OK at the bottom of the screen.
5. Now click the Home button in the toolbar.

Netscape Navigator 3.0:
1. In the Options menu, click General Preferences.
2. Click the Appearance tab.
3. In the Startup section, select Home Page Location.
4. In the text box immediately below Home Page Location, type http://www.boardpark.com.
5. Click OK.
6. Now click the Home button in the toolbar.
